# homeservices
The Home Services System is a platform that connects customers with professional service providers for household tasks such as cleaning, plumbing, electrical work, and repairs. Users can book services online, manage appointments, and review service providers, while providers can register and manage their service requests.

Tech Stack

-JavaScript
-Web technologies (HTML, CSS)
-Backend framework
-Django

Features
-Browse service providers
-Request home services
-Easy-to-use interface
